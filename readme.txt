Number System Converter

A little program to do some number system conversions.

changes
-	added AnyToDec-functionality

v0.2 AnyToDec
-	reads the number via java.util.Scanner
-	can convert a decimal number to any number system, the choosable base is 
	limited by 36 ("DecToAny")
-	can convert in the other direction: from a choosable base < 36 to decimal
-	see the umrechnung.odt for a mathematically example
-	watch "Symbol Table\SymTab.java" to see the assignment of numbers to 
	used number symbols
